# 🏦 Banking System

A console-based Banking System developed in **Python** that allows users to create bank accounts, securely log in, manage their finances, and perform banking operations. The project is built using Object-Oriented Programming (OOP) principles and stores data persistently using JSON.

---

## 📌 Features

### User Features
- ✅ Create a new bank account
- ✅ Auto-generated account numbers
- ✅ Secure 4-digit PIN authentication
- ✅ User login
- ✅ Deposit money
- ✅ Withdraw money
- ✅ Transfer money between accounts
- ✅ Check account balance
- ✅ View transaction history
- ✅ View account details

### Data Storage
- ✅ Persistent storage using JSON
- ✅ Automatic loading of existing accounts on startup
- ✅ Automatic saving after every transaction

## 🚀 Getting Started

### Prerequisites

- Python 3.10 or later
- PyCharm (recommended) or any Python IDE

### Clone the Repository

```bash
git clone https://github.com/omkantjha2006/Banking-System.git
```

### Navigate to the Project

```bash
cd Banking-System
```

### Run the Project

```bash
python banking_system.py
```
